# CYPRESS AUTOMATED TESTS

This project conatins automated Cypress Tests for SauceDemo.com

## Technologies Used

- Cypress
- Typescript

## Running Tests

### 1. Clone project
```
git clone https://github.com/dhodi-fahad/cypress-qa.git
```

### 2. Install Dependencies

```
cd cypress-qa 

npm install
```

### 3. Run tests
```
npm run test
```
